# 🌍 Travel Companion App

A mobile travel planning application developed using Flutter as part of the **SWT32042 – Practical for Mobile Application Development** module at **South Eastern University of Sri Lanka**.

---

## 📌 Overview

The **Travel Companion App** is designed to simplify and enhance the travel planning experience. It provides users with a centralized platform to explore destinations, organize trips, and manage travel activities efficiently.

---

## 🎯 Objectives

* Simplify travel planning through a single mobile platform
* Provide destination discovery with rich information
* Enable users to create and manage travel itineraries
* Offer a user-friendly and intuitive mobile interface

---

## 🚀 Key Features

### 🔐 Authentication

* Secure user registration and login
* Password validation and reset functionality

### 🏠 Home Dashboard

* Featured destinations
* Travel recommendations
* Category-based filtering (Beach, City, Adventure, etc.)

### 🔍 Explore Destinations

* Search destinations by keyword
* Filter by categories
* View ratings and destination details

### 📍 Destination Details

* Image gallery
* Description and key information
* Top attractions
* Bookmark / favourite option

### 📅 Trip Planner

* Create and manage trips
* Add destinations and travel dates
* Day-by-day itinerary planning
* Edit and delete plans

### 👤 User Profile

* View personal details
* Travel statistics (trips, countries, favourites)
* Manage saved destinations
* Edit profile and logout

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/hasmath27/MAD_Project.git
```

2. Navigate to the project directory:

```bash
cd MAD_Project
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the application:

```bash
flutter run
```
